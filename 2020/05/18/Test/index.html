There are at most kNk^NkN ways to partition NNN data points into clusters, so there are finite possible assignments. For each iteration of the algorithm,

  1. if the old assignment of centroids is same as the new, then the next clustering will keep same
  2. if the new assignment has lower distortion measure than the old one, then the old assignment will be replaced by the new one.

Since there is a finite number of possible assignments, each assignment has a unique minimum parameter of μk\boldsymbol{\mu}_kμk​, the K-meas algorithm will converge after a finite number of steps, when no re-assignment of data points will reduce the distortion measure. Also, if there are no different new assignment, μk\boldsymbol{\mu_k}μk​ will keep unchanged.
